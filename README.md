# PythonEdit
Pythonのコード専用のエディタです。
GUIにwxを使用しています。
動かなかったら入れてください。

## 開発言語
開発言語もPythonを使っています。
データはdataフォルダに保存されます。
## 使い方
まず,wxPythonライブラリを入れます。
### pip install wxPython
これでインストーるできない場合,多分パスが通ってないんだと思います。
なので
### py -m pip install wxPython 
でインストールできます。（pip自体インストールできてない場合もあるのでそこは各自でやってください）

後はこのプログラムを実行すればOKです。
